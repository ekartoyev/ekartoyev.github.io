Operators

If

The format is
if condition [then|goto|then goto] linenumber
Example:
50 if a<5 then 70
50 if a>5 goto 70
50 if a=5 then goto 70

Shortening operators

You can shorten the operators and write "ret" instead of "return" or "li" instead of "list". You should avoid shortening the operators too short as it may result in ambiguity. For example, if you write "l", the program will pick up "list" or "let" whichever comes first in its internal list of commands.
